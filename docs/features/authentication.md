# Authentication Feature

## Purpose

The authentication module provides a polished sign-in experience with a dedicated visual layout and theme switcher access.

## Route

- `/login`

## Included capabilities

- Branded auth background container.
- Centered login form card.
- Theme switcher directly on auth screen.
- Entry animation wrapper for smoother UX.

## Primary building blocks

- `src/app/(auth)/layout.tsx`
- `LoginForm`
- `AuroraBackground`
- `ThemeSwitcher`
- `AnimationWrapper`

## Notes

This module currently focuses on front-end UX scaffolding and can be connected to a backend auth provider in future iterations.
