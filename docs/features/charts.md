# Charts Feature

## Purpose

The charts module provides dashboard-grade visualization components for common analytics scenarios.

## Routes

- `/charts`
- `/charts/analytics`

## Included capabilities

- KPI summary cards.
- Line and area trend charts.
- Bar, pie, and radial charts.
- Reusable chart containers consistent with the dashboard theme.

## Primary building blocks

- `AnalyticsKpiCards`
- `RevenueLineChart`
- `VisitorAreaChart`
- `SalesBarChart`
- `TopProductsChart`
- `TrafficSourcesChart`
- `ConversionRadialChart`

## Business value

Offers production-ready visual patterns that teams can adapt to real business metrics with minimal UI rework.
