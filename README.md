# Starter Dashboard

A production-ready admin dashboard starter built with Next.js, shadcn/ui, and Tailwind CSS. It includes reusable modules for charts, tables, filters, forms, and settings — ready to extend for real products.

## Features

- **Dashboard** — KPI cards, analytics previews, and module navigation
- **Forms** — Rich text editor, image/file uploaders, and date inputs
- **Filters** — URL-synced search, select, tabs, date, pagination, and reset controls
- **Charts** — Recharts-based line, area, bar, pie, and radial visualizations
- **Table** — Sortable data tables powered by TanStack Table
- **Settings** — Dark theme customization with persistent presets
- **Auth** — Polished login screen with theme support

## Tech stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS 4 · shadcn/ui · Recharts · TanStack Table · TipTap

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
